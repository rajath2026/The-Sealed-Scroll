Rule I — Every letter walks backward

This scroll was written with a shift cipher. Each letter in the hidden message was pushed forward through the alphabet by a fixed number of steps. To read the truth, walk every coded letter backward by that same number. The alphabet loops — a letter stepping back past A wraps around to Z.

Rule II — The key hides in the answer

The shift is not given outright. Once decoded, the first word of the hidden message has exactly as many letters as the shift used to hide it. Use the dial to test shifts until the first word's own length agrees with the number you chose.
